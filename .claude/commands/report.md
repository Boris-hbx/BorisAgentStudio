# /report - 项目代码统计报告

生成当前项目的代码规模统计报告，包括各编程语言的代码行数、文件数量，以及工程文件（SPEC/RULE/STD/SKILL/Session）的数量统计。

## 执行步骤

1. 统计编程语言代码量（排除 node_modules, target, .git, dist 目录）：
   - TypeScript (.ts, .tsx)
   - CSS (.css)
   - Rust (.rs)
   - 计算各语言代码量占比

2. 统计工程文件数量：
   - SPEC 文档 (docs/specs/SPEC-*.md)
   - 规则文件 (rules/**/RULE-*.md)
   - 标准文件 (standards/**/STD-*.md)
   - 技能文件 (skills/**/SKILL-*.json)
   - Session 日志 (data/sessions/*.json)

3. 以并排表格形式输出结果：

```
📊 {项目名} 项目统计
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

编程语言                                        工程文件
┌──────────────┬────────┬────────┬──────────┐  ┌──────────────┬────────┐
│ 语言         │ 行数   │ 文件   │ 代码量占比│  │ 类型         │ 数量   │
├──────────────┼────────┼────────┼──────────┤  ├──────────────┼────────┤
│ TypeScript   │ {n}    │ {n}    │ {n}%     │  │ SPEC 文档    │ {n}    │
│ CSS          │ {n}    │ {n}    │ {n}%     │  │ 规则 (RULE)  │ {n}    │
│ Rust         │ {n}    │ {n}    │ {n}%     │  │ 标准 (STD)   │ {n}    │
├──────────────┼────────┼────────┼──────────┤  │ 技能 (SKILL) │ {n}    │
│ 合计         │ {n}    │ {n}    │ 100%     │  │ Session 日志 │ {n}    │
└──────────────┴────────┴────────┴──────────┘  └──────────────┴────────┘
```

## 格式要求

- 行数统计使用 `wc -l`
- 数字使用千分位格式（如 1,870）
- 占比保留一位小数
- 两个表格并排显示，便于一眼看清全貌
