# SPEC-010: 项目统计命令 (/report)

> 作者: Boris Huai
> 起草日期: 2026-01-25
> 状态: 已完成

## 概述

创建 Claude Code slash command `/report`，用于统计项目代码规模和工程文件数量。

## 命令调用

```
/report
```

## 统计内容

### 编程语言

| 语言 | 文件扩展名 |
|------|-----------|
| TypeScript | .ts, .tsx |
| CSS | .css |
| Rust | .rs |

### 工程文件

| 类型 | 路径模式 |
|------|----------|
| SPEC 文档 | docs/specs/SPEC-*.md |
| 规则 (RULE) | rules/**/RULE-*.md |
| 标准 (STD) | standards/**/STD-*.md |
| 技能 (SKILL) | skills/**/SKILL-*.json |
| Session 日志 | data/sessions/*.json |

### 排除目录

- node_modules/
- target/
- .git/
- dist/
